【*】 程序简介 
-工程名称：  402_Motor_PMSM_FOC_Sensorless_SpeedControl（无刷电机无感FOC）
-实验平台: 野火R7FA 6T2BD3CFP 启明 电机开发板


【 ！】功能简介：
使用启明6T2的ADC采集电机的三相电流，通用定时器输出PWM，然后通过瑞萨的电机库通过无感FOC的方法去控制直流无刷电机
来控制无刷电机启动、旋转方向、速度等等
学习目的：学会无刷电机的无感FOC。

【 ！】实验操作：
下载本程序，复位开发板，按键一按下启动电机（一直按下），电位器可调节电机转动方向和速度，

【 ！！】注意事项：
无

/*********************************************************************************************/

【*】 引脚分配

定时器TIM4，5，6输出6路PWM信号输出到PE10\PE11\PE12\PE13\PE14\PE15上连接到驱动板上的U+-\V+-\W+-接口.
通过ADC06、08、10采集三相电流 ，ADC01采集总线电，ADC15采集电位器

电机使能引脚:
电机驱动板 SD <----> 开发板 PB15
