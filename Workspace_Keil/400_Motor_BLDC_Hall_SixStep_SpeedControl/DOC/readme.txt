【*】 程序简介 
-工程名称：Motor_BLDC_Hall_SixStep_SpeedControl（无刷电机六步换向及速度调节）
-实验平台: 野火R7FA 6T2BD3CFP 启明 电机开发板


【 ！】功能简介：
使用启明6T2的通用定时器输出PWM，以控制直流无刷电机
来控制直流无刷电机启动、旋转方向、速度等等
学习目的：学会无刷电机的基本控制。

【 ！】实验操作：
下载本程序，复位开发板即可。

【 ！！】注意事项：
无

/*********************************************************************************************/

【*】 引脚分配

定时器TIM1输出PWM信号:
三路PWM输出到PE10\PE11\PE12上连接到驱动板上的U+\V+\W+接口.

电机使能引脚:
电机驱动板 SD <----> 开发板 PB15

编码器:
霍尔编码器信号从HU\HV\HW引脚输出,接入开发板PB06\PB07\PA10
